# skills_inbox — ResearchForge 搜集收件箱

把要纳入 ResearchForge 的 skill 丢到**本文件夹**即可。我（Claude）会在自动轮询时捡起来处理，**不需要你每次找我**。

## 怎么投递
- 一个 skill = 一个文件夹（内含 `SKILL.md` + 可选 references/scripts），整个拖进来；
- 或者单个 `.md` 文件、`.zip` 压缩包，也可以；
- 或者你只想给我一个已存在的路径，把路径写进 `_paths.txt`（每行一个）也行。

## 我会做什么（只读 / 不改你的原件）
1. 读每个 skill，提炼：**覆盖学科、能做的分析/模型、需要的数据前提、产出物**；
2. 登记进 `../docs/analysis-catalog/`（第②层"分析目录"草稿）；
3. 处理完把原件移到 `_processed/` 并在 catalog 里标注来源。

> 注：现在还在设计/搜集阶段，我不会动产品代码。catalog 只是知识库草稿。
