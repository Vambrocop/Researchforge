# meta_collection — 给"另一个 meta 项目"的暂存区

> 这里**不是 ResearchForge 的功能**。ResearchForge 本身不必深入做 meta 分析。
>
> 这是一个**暂存区**：当我在工作中遇到**与 meta（meta 分析 / meta 科学）相关的好内容**——方法、工具、数据集、论文、skill、代码片段——就丢到这里记一笔。用户日后会把它投喂给**另一个正在做的 meta 项目**，最终再决定怎么拼接。

## 怎么记
- 每条一行或一小段：**来源 + 一句话价值 + 链接/路径**。
- 大块材料放成单独文件，在本 README 留一行指针。
